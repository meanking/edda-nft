# Edda-NFT

## Using Software

NodeJS, Truffle

- Node v12.18.0
- Truffle 5.1.56
- Solidity - 0.6.6 (solc-js)

## Deploy Contract

```sh
truffle deploy --network [network_name]
```

## Verify Contract

```sh
truffle run verify [contract_name] --network [network_name]
```

## Using truffle console

```sh
truffle console
truffle develop
truffle develop --log
```

## Testing process

```sh
truffle test

```
